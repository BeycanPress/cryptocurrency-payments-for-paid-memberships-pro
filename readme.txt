== Description ==

First of all, we would like to point out that this plugin is not a plugin on its own. This plugin is a payment gateway developed for Paid Memberships Pro. In other words, with Paid Memberships Pro, users who want to sell memberships with crypto money payments over wordpress can use it. However, this plugin is an add-on for CryptoPay as well as a cryptocurrency payment gateway for Paid Memberships Pro. So the payment is actually CryptoPay, the main plugin that handles the cryptocurrency payment process. Therefore, users must have both plugins.

== Installation ==

1. Upload the entire `/cryptocurrency-payments-for-paid-memberships-pro` directory to the `/wp-content/plugins/` directory.
2. Activate CryptoPay Lite through the 'Plugins' menu in WordPress.
3. Enjoy.
